Testing your Installation

Once you've gone through the installation above, you can test your installation to see if most things have been installed correctly.

On the Mac

To run the test script, you will need a program called Terminal. To open your Mac's Terminal app, you can search for Terminal in Spotlight, or you can find it in the /Applications/Utilities folder. Once it's open, you should see something like: 

Last login: Thu Jun 20 13:31:37 on ttys000
ComputerName:~ yourname$

Now, download this file by right clicking the link and choosing "Save link as". Save the file to your home folder, and make sure it is called workshop_checklist.py .

Now, type python workshop_checklist.py after the $ in the Terminal, and press enter.

On the Command prompt type:
python workshop_checklist.py and press Enter
