Testing your Installation

Once you've gone through the installation above, you can test your installation to see if most things have been installed correctly.

On the Mac

To run the test script, you will need a program called Terminal. To open your Mac's Terminal app, you can search for Terminal in Spotlight, or you can find it in the /Applications/Utilities folder. Once it's open (and if you have Canopy installed and configured so it is available by default), you should see something like: 

Last login: Thu Jun 20 13:31:37 on ttys000
(Canopy 64bit) ComputerName:~ yourname$

Now, download the file workshop_checklist.py by right clicking and choosing "Save link as". Save the file to your home folder, and make sure it is called workshop_checklist.py .

Now, type python workshop_checklist.py after the $ in the Terminal, and press enter.

You should get an output like this:
